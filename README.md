## Objectives and Significance
The goal of this project is to predict whether a customer will respond to a marketing campaign using demographic, spending, and behavioral data. We treat this as a binary classification problem and plan to compare several machine learning approaches on the same dataset (such as logistic regression, decision trees, random forests, and neural networks). In addition to standard supervised learning models, we also want to study whether neural network optimization methods (e.g., different optimizers such as SGD and Adam) and unsupervised feature engineering (e.g., clustering with K-means and dimensionality reduction using PCA) can improve prediction performance.
This problem is meaningful because marketing campaigns are expensive, and companies often do not know which customers are most likely to respond. A more accurate prediction model can help businesses target users more effectively, reduce unnecessary outreach, and better understand different customer groups. We chose this topic because it combines practical value with several core ideas from machine learning, including classification, optimization, clustering, and dimensionality reduction.
Our main intuition is that we can learn different machine learning approaches. Since the dataset is relatively small and structured, model performance may depend not only on the classifier itself, but also on how the features are represented and how the learning process is optimized. By combining supervised learning with optimization and unsupervised feature discovery, we hope to better understand what leads to stronger and more reliable predictions.
